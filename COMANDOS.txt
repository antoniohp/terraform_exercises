IAC >> infraestructura como código.

los providers se comunican a través de las APIs de diferentes plataformas como azure, aws,GCP,kubernetes,github etc.

Para desplegar la infraestructura con Terraform:

Alcance - Identificar la infraestructura para su proyecto.
Autor - Escriba la configuración para su infraestructura.
Inicializar - Instalar los plugins que Terraform necesita para gestionar la infraestructura.
Planificar - Previsualizar los cambios que hará Terraform para ajustarse a su configuración.
Aplicar - Realizar los cambios planificados.




az login <<< para logearte en azure
terraform init para iniciar
terraform fmt para dar formato 
terraform validate para validar
terraform apply para aplicar y para aplicar cambios
terraform show para ver los cambios
terraform destroy para destruir todo
